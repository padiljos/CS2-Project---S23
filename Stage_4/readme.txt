INSTRUCTIONS TO RUN PROGRAM
1. Ensure all class files, including the csv files, are in the same folder
2. Open up command prompt with the path, to the class files, entered in
3. Compile Vet_ClinicGUI java source file using javac operator.
4. Run Vet_ClinicGUI file using java operator. This is the main program for the system, it will compile and run all other files along with it.
5. Follow instructions on each window to traverse the program. 
6. New users must create a profile before using the rest of the system. 
7. Existing users must enter userID to proceed.
8. Employees must enter password ("1661") to use the system.
