INSTRUCTIONS TO RUN PROGRAM
1. Ensure all class files are in the same folder.
2. Open up command prompt with the path, to the class files, entered in
3. Compile Vet_Clinic java source file using javac operator.
4. Run Vet_Clinic file using java operator. This is the main program for the system, it will compile and run all other files along with it.
5. Follow prompts on the screen to complete dummy info creation.
6. Once dummy info is created the main menu of the program will pop up allowing use of the system.
